About this Web Map

Welcome to the Arizona-Sonora Desert Museum Explorer!

  • Click around on the map to view the Desert Museum grounds.
  • Select one of the top boxes to filter descriptive points of plants, animals, or facilities.
  • Use the Locate button (top-left) to get your current location on the map.
  • Use the Home button (top-left) to return the map to default view.
  • Use the Editor button (top-left) to add Memory points to the map!
  • Use the Layers button (bottom-left) to view the legend and toggle layers in the map view.
